# Marq-TODOS

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm start
```

## Infomation

` @+id 넘버`를 통해 다른 Todo 리스트를 참조할 수 있습니다. 참조가 있는 list는 해당 list가 완료되어야 완료가 가능합니다.<br />
특정 list를 삭제할 경우 해당 list가 참조된 다른 list들의 참조 데이터가 지워집니다.
